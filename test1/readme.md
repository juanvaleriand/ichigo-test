## Run Project

Clone the project.

```bash
  git clone https://github.com/juanvaleriand/ichigo-test.git
```

Go to the project directory.

```bash
  cd ichigo-test
```

Install Node.js packages.

```bash
  npm install
```

Start the server.

```bash
  node app.js
```

Run unit test.
```bash
  npx mocha test
```
